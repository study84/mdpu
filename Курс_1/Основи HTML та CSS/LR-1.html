Лабораторна робота 1

HTML

HTML (Hypertext Markup Language) є мовою розмітки, яка використовується для створення структури та змісту веб-сторінок. Вона використовує теги (мітки), які вказують браузеру, як правильно відображати різні елементи, такі як текст, зображення, посилання та інші.

CSS

CSS (Cascading Style Sheets) є мовою стилів, яка використовується для оформлення та визначення зовнішнього вигляду веб-сторінок, створеної за допомогою HTML. CSS дозволяє вам контролювати розміщення, колір, шрифти та інші аспекти візуального представлення сторінок.

У поєднанні HTML та CSS створюють повноцінні веб-сторінки, де HTML відповідає за структуру та зміст, а CSS - за оформлення та стиль.
